## How GitHub Sync works

1. **Connect your GitHub account** on [neetcode.io/profile/github](https://neetcode.io/profile/github).
2. **Auto-commit** ŌĆö every time you submit a solution on NeetCode, it is pushed here automatically (configurable by status).
3. **Bulk Sync** ŌĆö push all your past solutions at once from the GitHub settings page.
4. **Manual sync** ŌĆö from the submission history panel on any problem page, sync or remove individual submissions.
